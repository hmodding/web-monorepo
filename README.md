# web-monorepo

- [web-frontend](web-frontend/README.md)
- [web-backend](web-backend/README.md)

## Requirements

- npm 7 (or higher)
- node v16 (or higher)

## Get started

### build

```
npm run build
```

### start

```
npm run start
```

### dev

```
npm run dev
```
