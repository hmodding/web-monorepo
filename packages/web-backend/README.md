# web-backend

## Requirements

- [nodejs & npm](https://nodejs.org/en/)
- [typescript](https://www.typescriptlang.org/)
- [raftmodding web-frontend](https://github.com/raftmodding/web-frontend)

## Get started

### dev

```shell
npm run dev
```

### build

```shell
npm run build
```

### apply code-style

```shell
npm run format
```
