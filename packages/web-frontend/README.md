## Requirements

- [nodejs & npm](https://nodejs.org/en/)
- [vite](https://github.com/vitejs/vite)
- [typescript](https://www.typescriptlang.org/)
- [raftmodding web-backend](https://github.com/raftmodding/web-backend)

## Get started

### dev

```shell
npm run dev
```

### build

```shell
npm run build
```

### apply code-style

```shell
npm run format
```
